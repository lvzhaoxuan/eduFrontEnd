# 合约相关接口文档
---
## 1.  创建电子学历证接口
### 1.1  功能描述
创建电子学历证
### 1.2  请求说明
+ 请求方式：POST
+ 请求URL地址：<http://localhost/evidenceCreate>
### 1.3  请求参数
| 字段 | 字段类型 | 字段说明 |
| ---- | ---- | ---- |
| stuId | 字符串 | 学生身份证号 |
| schNo | 字符串 | 学校编号 |
### 1.4  返回结果
    CommonResponse(code=200, message=200, data=ok)
### 1.5  错误状态码
| 错误状态码 | 说明 |
| ----  | ---- |
| 50002 | 创建合约失败 |
