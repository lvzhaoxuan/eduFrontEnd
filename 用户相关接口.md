# 用户相关接口文档
---
## 1.  学生注册接口
### 1.1  功能描述
学生进行注册
### 1.2  请求说明
+ 请求方式：POST
+ 请求URL地址：<http://localhost/stuRegister>
### 1.3  请求参数
| 字段 | 字段类型 | 字段说明 |
| ---- | ---- | ---- |
| address | 字符串 | 区块链账户地址 |
| password | 字符串 | 密码 |
| name | 字符串 | 姓名 |
| sex | 字符串 | 性别 |
| birthDate | 字符串 | 出生日期 |
| idCard | 字符串 | 身份证号 |
| admDate | 字符串 | 入校日期 |
| graduDate | 字符串 | 毕业日期 |
| major | 字符串 | 所学专业 |
### 1.4  返回结果
    {"code":"200","message":"200","data":"ok"}
### 1.5  错误状态码
| 错误状态码 | 说明 |
| ----  | ---- |
| 50001 | 注册失败 |
